# Contributing

Thanks for helping improve the AI SEO / GEO skill. Issues and pull requests are welcome.

## Ways to contribute

- **Corrections** — anything inaccurate, outdated, or unclear.
- **Fresh research** — newer citation studies or platform behavior. **Date every figure** and cite the source; this skill treats statistics as perishable.
- **New verticals / language playbooks** — the MENA / Arabic reference is a model for others.

## Ground rules for skill content

1. **No promotional tone** ("best", "leading", "act now" suppress citation — practice what the skill preaches).
2. **Self-contained passages** — every claim reads correctly out of context.
3. **Date and source statistics**, and frame them as perishable.
4. **Don't fabricate** — if uncertain, say so.

## Submitting

1. Fork, branch, make your change. If you edit the skill, keep the `references/` cross-links valid.
2. Open a PR describing **what** changed and **why**, with sources for any new figures.
