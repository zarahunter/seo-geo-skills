# Code of Conduct

This project follows a simple standard: **be respectful, be constructive, assume good faith.**

- Harassment, discrimination, and personal attacks are not tolerated.
- Critique ideas and content, not people.
- Keep discussion relevant and professional.

Maintainers may edit, remove, or reject comments, commits, issues, and pull requests that violate this standard.

Report concerns by opening a private issue or contacting the maintainer via [@zarahunter](https://github.com/zarahunter).
